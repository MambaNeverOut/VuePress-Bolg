# 前端词汇

- file 文件
- link 链接
- make 制作
- find 找到
- move 移动
- echo 回声
- remove 删除
- touch 触摸
- copy 拷贝
- change 改变
- list 列表
- directory 目录/文件夹
- recursive 递归的
- force 强制
- heading 标题
- order 顺序，秩序
- ordered 有顺序的
- paragraph 段落
- unordered 无顺序的
- description 描述
- term 术语
- aside 旁边的
- quote 引用
- anchor 锚，定点
- emphasis 强调，重读
- break 打断
- hyper 超级
- blank 空白
- target 目标
- reference 引用
- self 自己
- frame 边框、框架
